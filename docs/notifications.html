Notifications & Alerts

Notifications Module

The Notifications module delivers real-time system alerts and SMS messages related to reservations, task assignments, approvals, status changes, and administrative actions to ensure timely communication and operational awareness.

Overview Notification Types Accessing Notifications System Behavior
OVERVIEW

Real-Time Alerts and Messaging

  • Provides real-time system notifications for reservations, task assignments, approvals, and updates.
  • Ensures staff and administrators are immediately informed of actions requiring attention.
  • Supports both in-system alerts and SMS notifications for critical operational updates.
  • Improves response time, accountability, and coordination across system users.
CATEGORIES

Types of Notifications

  • Reservation Notifications – Alerts for reservation approvals, rejections, cancellations, and schedule updates.
  • Task Assignment Notifications – Sent when tasks are assigned, updated, completed, or overdue.
  • Status Change Alerts – Notifies users of progress updates or administrative actions.
  • SMS Notifications – Critical alerts sent via SMS to assigned staff for immediate attention.
STEP-BY-STEP

Accessing Notifications

1

Log In to the System

User logs in using their authorized account credentials.

2

Locate Notification Icon

The notification bell icon appears on the system header when new alerts are available.

3

View Notifications

Clicking the icon displays a list of recent notifications with timestamps and brief descriptions.

4

Open Notification Details

Selecting a notification redirects the user to the related reservation, task, or system record.

SYSTEM LOGIC

Notification Behavior

  • Notifications are automatically generated based on system actions and administrative triggers.
  • SMS notifications are sent for critical task assignments and time-sensitive updates.
  • Administrators can verify actions through notification logs and system records.
  • Notifications remain accessible for reference and auditing purposes.

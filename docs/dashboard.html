Administrator Dashboard

The Administrator Dashboard provides system-wide visibility over reservations, equipment usage, approvals, and operational risks.

Displays the total number of registered system users.

Pending Requests

Shows reservation requests awaiting approval.

Approved Requests

Active Reservations

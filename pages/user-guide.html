OVERVIEW

About TechReserve Users

TechReserve users include students, faculty members, student officers, and other authorized personnel who require access to institutional equipment, facilities, and resources. Through the platform, users can conveniently submit reservation requests without the need for manual paperwork.


Through a centralized reservation platform, users can easily determine equipment availability, submit requests, and receive updates regarding approvals and borrowing schedules.

USER PERMISSIONS

User Capabilities

Authorized User

  • View equipment availability schedules

  • View reservation history records
  • Receive reservation notifications
  • Manage personal account information
  • Cancel pending reservations when permitted

USER ACCESS

Create a User Account

1

User Signs Up

A new user initiates account creation by completing the Sign Up process in TechReserve using valid personal and institutional information.

2

Account Added to Wishlist

Upon successful sign up, the user account is temporarily stored in the Wishlist pending administrative review.

3

Admin Verification

An administrator reviews the user’s submitted information and verifies eligibility for system access.

4

Approval or Denial Decision

If the request is denied, the account is stored in the database with a Denied status.

If approved, the administrator proceeds to send an account invitation.

5

Send User Invitation

The system sends an invitation to the verified user, allowing them to activate their account.

6

User Accepts Invitation

Once the invitation is accepted, the user’s account status is updated in the User Accounts Database, granting access to the system.

7

Access Dashboard

The user signs in and is redirected to the Dashboard, confirming successful account activation.

8

Update Account Information

From Account Settings, users may update personal information and credentials.

Role changes and account status modifications remain restricted to administrators.

9

Account Deactivation or Deletion

User accounts may be deactivated based on system rules or deleted by administrators when necessary, with all actions reflected in the database.

STEP-BY-STEP

How to Create a Reservation

1

Sign In to Your Account

Log in using your registered institutional email address and password to access the TechReserve dashboard.

2

Go to Reservations

From the dashboard, navigate to the Reservations section to start a new request.

3

Enter Reservation Details

Provide the purpose of use, reservation date, time slot, and other required information.

4

Select Venue or Equipment

Choose the facility or equipment you want to reserve. Availability indicators will help you select valid options.

5

Add Additional Requirements

Include any special requests or notes needed for the reservation.

6

Submit the Reservation

Review all details carefully, then submit the request for administrator approval.

7

Track Request Status

Monitor the reservation status under Pending Requests until approval.

POLICIES

Account Rules

Valid Email

Users must register using an active institutional email account.

Password Security

Passwords should remain confidential and must not be shared with other users.

Personal Responsibility

Users are responsible for all reservations made under their account.

Misuse Prevention

Providing false information or abusing reservation privileges may result in account suspension.

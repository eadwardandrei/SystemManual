OVERVIEW

Administrator Responsibilities

Administrators are responsible for overseeing the entire TechReserve system. Their role includes managing user accounts, monitoring reservation activities, approving or rejecting requests, and maintaining accurate equipment records.


Proper administrative control ensures fair resource allocation, system security, and smooth operational workflows across all departments.

ACCESS CONTROL

Administrator Account Access

Administrator accounts are created and authorized by system owners or super administrators. Elevated privileges allow full access to management and configuration tools.

Admin credentials must be protected to prevent unauthorized system changes or data misuse.

ADMIN PERMISSIONS

Administrator Capabilities

System Administrator

  • Approve or reject reservation requests
  • Manage equipment inventory
  • Create, update, or deactivate user accounts
  • Monitor reservation logs and history
  • Configure system rules and limits
  • Generate reports and summaries

ADMIN ACCESS

Create an Administrator Account

1

Log in to the Administrator Account

An existing administrator logs in to TechReserve using valid admin credentials to access system management features.

2

Access the Requests Hub

From the Dashboard, navigate to the Requests Hub where account-related actions and invitations are managed.

3

Open the Add Admin Account Modal

Select the Add Admin Account option to open the account creation modal for administrators.

4

Send an Admin Invitation

Using the Admin tab, enter the required information and send an invitation to the new administrator.

5

Monitor Invitation Status

The invitation appears in the View Invite Account modal within the Requests Hub, displaying its current status.

6

Invitation Accepted or Expired

If the invite is accepted, the request badge is updated to Accepted and the administrator account is added to the Manage Accounts table.

If the invite is not accepted within 7 days, it is marked as Expired and requires re-sending.

ACCOUNT MANAGEMENT

Update and View Accounts

User Account Management

1

Log in to the System

The user logs in to TechReserve using valid account credentials.

2

Access Account Settings

From the Dashboard, navigate to Settings to view personal account information.

3

Update Own Account Details

Users may update editable account information such as profile details and credentials.

Role assignment and account deactivation are restricted to administrators.

4

Save Changes and Log Out

Once updates are saved, the user may log out to complete the session securely.

Administrator Account Management

1

Log in as Administrator

The administrator logs in to TechReserve using verified admin credentials.

2

Open Manage Accounts Page

From the Dashboard, navigate to the Manage Accounts page to view all registered users.

3

View and Update User Accounts

Administrators may perform the following actions:

  • Deactivate or reactivate user accounts
  • Update account information and roles
  • Delete accounts when necessary
4

Confirm Changes and Log Out

All updates are reflected in real time within the system. The administrator logs out after completing account management tasks.
